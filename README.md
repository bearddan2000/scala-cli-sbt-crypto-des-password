# scala-cli-maven-crypto-des-password

## Description
A demo for DES encryption of a password.
DES is a 48 byte encryption.

## Tech stack
- scala
- maven

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
